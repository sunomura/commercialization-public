# <span id="p_hlk_test.818bdd92-529a-439e-a7f1-f4a74cd2b84c"></span>NDISTest 6.0 - \[1 Machine\] - 1c\_KernelCalls


This test discovers which kernel calls a driver is making and then checks to see if they are legal. In some of the cases of illegal calls, the log file will also contain information on which legal functions can be substituted for illegal function call.
