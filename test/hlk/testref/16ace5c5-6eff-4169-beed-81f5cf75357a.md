# <span id="p_hlk_test.16ace5c5-6eff-4169-beed-81f5cf75357a"></span>System Audio - AC3 Test

>[!IMPORTANT]
>  
This test requires supplemental content. You can find this content in the 'Windows HLK supplemental test content' section at the following location:

-   <http://go.microsoft.com/fwlink/p/?LinkId=532718>

 

This test verifies that audio devices that support the AC-3 format comply with the Hardware Certification requirements.

>[!NOTE]
>  
This is a system-specific implementation of an existing test. For more information on this test, see [AC3 Test - Certification](b42bd7fc-32b1-4500-8049-a4e00c70eb33.md).
