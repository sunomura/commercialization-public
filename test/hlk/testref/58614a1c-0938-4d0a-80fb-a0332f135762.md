# <span id="p_hlk_test.58614a1c-0938-4d0a-80fb-a0332f135762"></span>WDDM GammaRamp


This automated test verifies that kernel-mode video drivers succeed in attempts to set gamma ramps.

This test is a basic test of Longhorn Display Driver Model (LDDM) gamma-ramp functionality. It sets gamma ramps by using Graphics Device Interface (GDI) and kernel-mode entry points for the operating system. Drivers must not fail during this process.
